## 安装

### 安装依赖
```bash
npm install
```
### 构建程序
```bash
npm run build 
```

## 贡献代码

### 第一步： fork 仓库

将 snow fork 到您自己的代码仓库

### 第二步： clone 您自己仓库的代码

```bash
git@github.com:guoshijiang/snow.git
```

### 第三步：建立分支编写提交代码

```bash
git branch -C xxx
git checkout xxx
编写您的代码
git add .
git commit -m "xxx"
git push origin xxx
```

### 第四步：提交 PR

到你的 github 上面有一个 pr, 提交到 snow 代码库


### 第五步：review 完成

待 snow 代码维护者 review 通过之后代码会合并到 snow 库中，至此，您的 PR 就提交完成了
